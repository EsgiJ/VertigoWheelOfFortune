# Wheel of Fortune Vertigo Games Demo

A wheel of fortune gambling game built for the Vertigo Games New Grad Developer testcase.

## Download

Get the APK from releases

## How to Play

1. Spin the wheel to win rewards
2. Each spin advances you to the next zone
3. Hit a bomb on a normal zone and lose all rewards (or spend gold to revive)
4. Every 5th zone is safe, no bomb, can cash out
5. Every 30th zone is super, golden wheel with special rewards
